Caesar cipher (JavaScript)

Program

Description

This JavaScript application is designed to perform message encryption and decryption using the Caesar cipher. The user must provide an integer as the shift value, which should range from 0 to 25, both inclusive. By default, a shift value of 13 corresponds to the ROT13 cipher. The program ensures that the letter casing remains unchanged during the encryption or decryption process, and any non-letter characters remain unaltered.
